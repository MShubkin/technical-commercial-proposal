# Модуль "Технико-коммерческое предложение"

## Описание

Функциональность сбора ТКП является отдельным модулем АСЭЗ 2.0.

Перечнь функций модуля:

- Создание ЗЦИ
- Отправка ЗЦИ на внешний сервис
- Подача ТКП к ЗЦИ
- Проверка ТКП
- Завершение ЗЦИ
- Контроль ответов на ЗЦИ
- Отмена ЗЦИ
- Уведомления
- Поиск ЗЦИ и ТКП
- Дополнительные настройки списков
